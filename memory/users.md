Primary User: Distributor (Owner)

Description:
The user visits shops daily and records orders and payments.

Requirements:

* The UI must be extremely simple
* Must work well on a mobile phone
* Large buttons and minimal navigation
* Fast shop search
* Quick order entry
* Quick payment entry

Usage Context

The user often uses the app while standing in shops and talking to customers.
Actions must be quick and require minimal typing.
The interface should prioritize speed and clarity over complex features.
